📖 技术栈术语表

快速查阅 · 按字母排序
API
应用程序接口。本项目中 Pipeline 通过 API 调用 DeepSeek 的 LLM 服务来生成摘要。
AIGC
AI Generated Content，AI 生成内容。本项目每篇文章底部标注"AI 辅助生成"以满足合规要求。
Build（构建）
Hugo 把 .md 文件 + HTML 模板"编译"成最终网站的 HTML 文件的过程。命令：hugo --gc --minify。
CDN
Content Delivery Network，内容分发网络。Cloudflare Pages 提供 CDN 加速，让全球读者都能快速访问。
CI/CD
持续集成/持续部署。本项目的 GitHub Actions 工作流：每天自动跑 Pipeline → 构建 Hugo → 部署到 Cloudflare。
Cloudflare Pages
静态站点托管平台。本项目博客最终部署的目标。免费版提供无限带宽。
Content Collection
Hugo 的概念（注意：本项目用的是 Hugo 标准版，不是 Astro）。Hugo 从 content/ 目录读取 .md 文件生成页面。
CSS Pipes（Hugo Pipes）
Hugo 内置的资产处理机制。本项目用 Hugo Pipes + PostCSS 处理 Tailwind CSS。
DeepSeek API
本项目的 LLM 提供商（替代了最初的 Claude）。Pipeline 通过 OpenAI 兼容接口调用 DeepSeek 做新闻摘要和相关性判断。
Embedding
将文本转换为向量（数字数组）的技术。本项目用 sentence-transformers 做摘要与原文的相似度校验。
Frontmatter
.md 文件开头的 YAML 元数据区（两个 --- 之间的内容）。Hugo 通过 frontmatter 获取文章的标题、日期、分类等信息。
GitHub Actions
GitHub 的 CI/CD 服务。本项目用它做两件事：每天早上 06:00 定时跑 Pipeline，以及提交代码时自动部署。
Go Template
Hugo 模板使用的模板语言。比如 {{ .Title }} 输出文章标题，{{ range .Pages }} 遍历文章列表。
Hugo
基于 Go 语言的静态站点生成器。速度极快，单二进制文件。本项目的博客框架。
LLM
Large Language Model，大语言模型。Pipeline 使用 DeepSeek（具体是 deepseek-chat）做新闻摘要和过滤器。
Markdown
轻量级标记语言。项目中的文章以 .md 格式存储，Hugo 将其渲染为 HTML 页面。
Pipeline
管道/流水线。本项目中指 pipeline/ 目录下的 Python 程序：采集 → 过滤 → 摘要 → 发布。
PostCSS
CSS 后处理器。Hugo 用 PostCSS + Tailwind 插件来处理 CSS。
RSS
Really Simple Syndication，一种内容订阅格式。本项目的博客提供 /feed.xml，读者可以用 RSS 阅读器订阅。
Static Site Generator (SSG)
静态站点生成器。Hugo 就是一款 SSG——它在构建时生成所有页面，而不是用户访问时才动态生成。优点是速度快、成本低。
Tailwind CSS
工具类优先的 CSS 框架。本项目使用 Tailwind v4，通过 PostCSS 集成到 Hugo 中。
Venv（Virtual Environment）
Python 虚拟环境。本项目的 Python 依赖安装在 pipeline/.venv/ 中。
YAML
一种配置文件格式。项目中多处使用：文章的 frontmatter、config.yaml（Pipeline 配置）、hugo.toml（站点配置）。
