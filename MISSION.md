# Mission: AI 新闻早报平台技术栈

## Why

我正在构建一个自动化 AI 新闻早报平台。技术栈涉及 Hugo、Python、Tailwind CSS、CI/CD 等多个领域。我需要理解整个技术栈如何配合工作，以便后续能独立维护、调试和扩展这个系统。

## Success looks like

- 能说清博客（Hugo）、管道（Pipeline）、部署（CI/CD）三层各自做什么
- 能看懂和改 Hugo 模板（Go Template 语法）
- 能看懂和改 Pipeline 的 Python 代码
- 能独立调试部署问题

## Constraints

- 个人项目/副业，成本敏感
- 对 TypeScript 不熟悉，但对 Python 有一定了解
- 之前在终端操作过一些命令

## Out of scope

- 深入学习 Python 语言本身
- 深入学习 Go 语言（仅限 Hugo 模板层面）
- CSS/前端设计精修
